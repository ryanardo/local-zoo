## SETUP/INSTALLATION REQUIREMENTS
### NPM Packages
```
npm install
```
### Bower
```
bower install
```
### Gulp
Build & Serve
```
gulp build; gulp serve
```
### Testing
```
npm test
```

#### Quick run/setup
If you want to just get the project to run, try this command!
```
npm install; bower install; gulp build; gulp serve
```
